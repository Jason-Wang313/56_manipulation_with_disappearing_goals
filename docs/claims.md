# Claims

## Supported

- Persistent goal proxies can preserve task identity in a synthetic disappearing-goal diagnostic.
- Re-binding and re-acquisition must be explicit parts of the proxy interface.
- Identity swaps are a core metric, not a secondary perception detail.

## Not Supported

- Deployment-ready manipulation.
- Universal solution to partial observability.
- Silent nearest-neighbor proxy acceptance under close distractors.

## V2 Boundary

Loose proxy acceptance collapses under close distractors and severe ambiguity. Proxy re-acquisition helps but still has nonzero swap rate under severe ambiguity.
